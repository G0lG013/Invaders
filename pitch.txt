Game Idea:
The game will be in vertical scrolling World in space. The game will consist different stages and Boss.

Character Desciption:
1. The main character will be some kind of small spaceship.
2. Enemies such as alien space ships that's targeting the main character
3. There will be different types of Boss in different stages.


Win condition: The main Character must shoot the enemies in order to survive, to get points and to proceed to another stages.

Lost Condition: The main character dies if it touches alien space ship, boss or their bullets.


Inspiration:
This game was inspired by a video game called striker 1945.
